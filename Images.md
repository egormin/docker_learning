### Docker images
Имэйджи можно посмотреть и скачать с сайта: `https://hub.docker.com/explore/`.

***Для поиска нужного имэйджа*** используется команда:
```
docker search <name>
```

***Посмотреть список имэйджей в системе:***
```
docker image ls
```

***Удалить имэйдж:***
```
docker image rm <name>
```
